# Pagina-HTML-Semantica
Este projeto consiste na criação de uma página web em HTML5, desenvolvida com o objetivo de praticar e consolidar o uso das principais tags aprendidas em HTML.
A estrutura da página utiliza HTML semântico, incluindo tags como header, nav, main, section, article, aside e footer, garantindo melhor organização, acessibilidade e leitura do código.

## O projeto também explora:
- Títulos hierárquicos (h1 até h6)
- Parágrafos, span e div
- Links e navegação
- Tabelas para organização de dados
- Elementos de mídia como video, audio e iframe
- Formulários com campos de entrada
- Boas práticas de estruturação HTML

 Essa página foi criada com foco educacional, servindo como base para estudos futuros em CSS e JavaScript, além de ajudar no entendimento da estrutura fundamental da web.
